- MySQL (See [MySQL Installation](https://dev.mysql.com/doc/mysql-installation-excerpt/5.7/en/))


### Installation
- Clone this repo

    ```sh
        git clone https://github.com/tusfendi/web_sales.git
    ```

- Copy `.env.example` to `.env`

    ```sh
        cp .env.example .env
    ```
- Setup local env
- Setup local database using db migration

- Start service
    ```sh
        php artisan serve
    ```
